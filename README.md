Mudah
1. Pertama program memerintahkan kita untuk memasukkan jarak yang akan ditempuh oleh barang, jika <=10 KM maka akan dikenai harga 4250 setiap KG barang muatannya, dan jika >10KM maka akan dikenai harga 6000/KGnya. Lalu program akan menyuruh kita memasukkan panjang, lebar, dan tinggi dari barang yang kita muat, jikalau volumenya melebihi 100cm3 maka akan terkena tambahan harga sebesar 50000

2. Program akan memerintahkan kita memasukkan sebuah angka, jikalau angka tersebut habis dibagi 2 maka angka tersebut adalah angka genap, dan jika tak habis dibagi 2 maka adalah bilangan ganjil


Sedang
1. Program akan memerintah kita untuk memasukkan angka, dan akan dibuatkan faktorialnya. Variabel angka tersebut akan dikurangi menggunakan for dan setiap angka akan dikali lalu muncullah hasil faktorial tersebut

2. Program akan memerintahkan kita memasukkan jari-jari dan tinggi dari tabung yang akan kita hitung, lalu program akan menghitungnya dengan rumus volume tabung (Pi*jari^2*tinggi)

Susah
1. Program akan memeirntah kita untuk memasukkan jumlah siswa, dan menggunakan perulangan for sebanyak jumlah siswa dan pada setiap pengulangan kita disuruh menginput nilai siswa tsb. Lalu setiap nilai siswa dijumlah dan dibagi dengan jumlah siswa

2. Pada array pertama dan kedua kita disuruh untuk menginput jumlah kolom dan baris untuk matrix yang akan dibuat, lalu kita disuruh menginput nilai pada setiap kolom dan baris menggunakan metode for, lalu hasil akan ditampilkan menggunakan metode for juga
